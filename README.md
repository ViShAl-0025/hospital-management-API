# Hospital Management API

A REST API built using Node.js, Express.js and MongoDB Atlas.

## Features

- Add Patient
- Get All Patients
- Update Patient
- Delete Patient

## Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Postman

## API Endpoints

POST /api/patients

GET /api/patients

PUT /api/patients/:id

DELETE /api/patients/:id
