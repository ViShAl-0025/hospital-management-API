[# Hospital Management API

A RESTful API built using Node.js, Express.js, MongoDB Atlas and Mongoose for managing patient records.

## Features

* Add Patient
* Get All Patients
* Update Patient Details
* Delete Patient
* MongoDB Atlas Integration
* REST API Architecture

## Tech Stack

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* dotenv

## API Endpoints

### Add Patient

POST /api/patients

### Get All Patients

GET /api/patients

### Update Patient

PUT /api/patients/:id

### Delete Patient

DELETE /api/patients/:id
